# 🍕 The Pizza Plug

Welcome to **The Pizza Plug**—a sleek, high-energy, and responsive web application designed for a premium modern pizza experience. Built from the ground up using a modern frontend ecosystem, this project focuses on fast performance, clean typography, and a striking visual layout.

## 🚀 Tech Stack

*   **Framework:** React 18 (TypeScript)
*   **Build Tool:** Vite (Optimized asset bundling)
*   **Styling:** Tailwind CSS v4.0 (Utilizing modern, CSS-first `@theme` configurations)
*   **Typography:** Custom display configurations featuring *Bebas Neue*, *Italiana*, and *Syne* fonts.
